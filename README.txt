             ИНСТРУКЦИЯ ПО УСТАНОВКЕ / RUN ИНФО
========================================================================

Это официальный репозиторий программы AudioEditorV2 от VodPlay. 
Ниже представлена простая инструкция, как запустить и установить софт.

------------------------------------------------------------------------
 КАК УСТАНОВИТЬ ПРОГРАММУ
------------------------------------------------------------------------
 1. Скачайте файл `AudioEditorV2Installer.html` из этого репозитория 
    (или откройте его, если вы уже скачали проект).
 
 2. Запустите файл `AudioEditorV2Installer.html` через любой удобный вам 
    браузер (Google Chrome, Яндекс.Браузер, Opera, Edge и т.д.).
 
 3. Нажмите на большую яркую кнопку «Скачать программу» в центре экрана.
 
 4. Начнется автоматическая загрузка официального ZIP-архива 
    с программой напрямую из защищенного облачного хранилища (DropBox).
 
 5. После завершения загрузки распакуйте архив `AudioEditorV2.zip` 
    в любую удобную для вас папку на компьютере.
 
 6. Запустите файл `AudioEditorV2.exe`. Программа готова к работе!
    (Внутри архива также будет лежать подробный README.txt с описанием 
    всех функций и настроек звука).
